    Learn to code the CodeKids way

    Code is a language

    Learning to program is just like learning any language. The earlier you start, the easier it is. Kids can use CodeKids as soon as they can read and write.

    Short and sweet

    If you want your kid to learn guitar, you don't start with Rock Band. You start with a smaller guitar. CodeKids uses simple commands to create cool apps that really work.

    Monkey see, monkey do

    Most people learn hard things by imitation and repetition. With CodeKids, kids learn to program by copying apps and making them their own.
